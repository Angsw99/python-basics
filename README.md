# python-basics
Repository to revise my Python basics, including the use of numpy, matplotlab, etc.

## Contents
**Sine and Cosine Plots**  
   File: 'py scripts/01_sine_cosine.py'  
   Demonstrates plotting sine, cosine, and their squares on subplots.  
   Libraries used: numpy, matplotlib
   ![Sine and Cosine Plots](output/sine_cosine_plots.png)

**Show Numpy Array as an Image**  
   File: 'py scripts/02_numpy-imshow.py'  
   Demonstrates creating a numpy array with zeroes, changing the values and plotting the values as an image.  
   Libraries used: numpy, matplotlib
   ![Numpy Array As an Image](output/numpy_imshow.png)

**Random Samples and Histograms (Central Limit Theorem)**  
  File: 'py scripts/03_random-histogram.py'
  Generate random samples from a normal distribution, find mean of each sample and plot the sample means of each histogram. As the sample size increases, the distribution of the sameple mean will be closer to a normal distribution.
  Libraries used: numpy, matplotlib
  ![random_histogram.png](output/random_histogram.png)
